## Installation Instructions
```bash
  git clone http://github.com/kunley247/phishsighter
  cd phishsighter
  python -m env myvenv
  source ./myvenv/Script/activate
  pip install -r requirements.txt
```

## Usage Instructions
1. For the Web App
  ```bash python phishsighter.py --web ```
